# 镜头规则

本文件在成片路线和视觉风格之后调用。先确定 `vertical-short`、`horizontal-short` 或 `horizontal-film`，再选择对应的景别、运镜、镜头控制、焦距和光学特效。竖版镜头和横版镜头不要混用。

运镜必须服务剧情信息、人物关系或情绪变化。写 `video_prompt` 时，不要只写“镜头移动”；必须写清楚“景别 + 运镜手法 + 速度/幅度 + 运动终点/构图落点 + 主体动作/反应 + 叙事目的”。单镜头原则上只使用一种主运镜，最多叠加一个轻微辅助动作，例如“慢速推进 + 焦点切换”。

## 运镜库强制调用规则

- 每个有运动的镜头，必须从下方 `AI运镜手法库（38种）` 中选择一个手法原名写入镜头描述，例如 `慢速推进`、`快速甩镜`、`过肩镜头`、`焦点切换`、`跟随式跟拍`。
- 运镜字段必须写成：`<景别，AI运镜手法库原名 + 速度/幅度 + 运动终点/构图落点，焦距。画面描述。>`；不强制写“摄像机起始位置”。若该库项本身包含速度词，例如 `慢速推进`、`快速甩镜`，仍要补充幅度或落点，例如“慢速推进小幅靠近到@角色脸部近景”。固定镜头写“固定镜头/固定特写（静态例外），画面保持稳定”。
- 不得只写非库泛称，例如“慢推、微推、推近、拉远、摇镜、甩镜、手持微晃、近距离跟拍、固定镜头、平移跟拍”。必须改成库内原名：慢推/微推/推近 -> `慢速推进` 或 `平滑向内变焦`；拉远 -> `慢速后拉` 或 `平滑向外变焦`；甩镜 -> `快速甩镜`；手持微晃 -> `手持纪录片`；近距离跟拍 -> `引导式跟拍`、`跟随式跟拍` 或 `平行侧跟`；平移 -> `横向左移` 或 `横向右移`。
- 静态镜头例外：只有人物口型、微表情、道具读数或关键反应需要稳定观看时，才可写 `固定镜头/固定特写（静态例外）`。静态例外不属于AI运镜手法库，同一组最多2次；除静态例外外，每个镜头都必须使用库内原名。
- 每个运动镜头都必须写运动终点或落点。终点可以是最终构图主体、最终画面尺度、动作落点或焦点落点，例如“横向右移到门框左侧露出@角色肩背”“垂直下移落到桌面信封特写”“焦点切换从前景文件落到@角色侧脸”。固定镜头必须说明画面保持稳定。
- 组合手法最多两个，且第二个只能是轻辅助，例如 `过肩镜头 + 焦点切换`、`慢速推进 + 焦点切换`。不得堆叠三个以上运镜。
- 人物近景、特写、大特写使用 `慢速推进`、`快速推进`、`平滑向内变焦`、`急速变焦` 时，必须明确人物不是纯正脸看镜头；两人单独对话优先写轻微侧角、微侧脸或侧脸，并写清视线落点是对手、文件、道具、门口、地面或画外目标。只有剧本明确自拍视频、直播、监控口供或第一人称主观对视时，才允许写看向镜头。

## AI运镜手法库（38种）

以下手法用于生成 AI 视频提示词时选择。不要机械堆叠；先判断场景意图，再选择最少、最准确的运镜。

1. 慢速推进：镜头缓慢靠近主体，用于情绪聚焦、压力递增、人物即将说出关键信息。
2. 慢速后拉：镜头缓慢远离主体，用于揭示环境、孤立感、关系疏离或事件后果。
3. 快速推进：镜头突然逼近主体，用于爆点台词、惊觉、威胁、身份揭露。
4. 眩晕效应：轨道前后移动配合反向变焦，用于恐惧、失控、心理震荡；真人短剧慎用。
5. 微距变焦：从细节进入更微观的纹理或物件，用于眼睛、血迹、戒指、手机信息、证据。
6. 超远距变焦：从远处快速或平滑压到目标，用于大空间定位、史诗开场或远处人物被发现。
7. 过肩镜头：从一人肩后看另一人，用于对话、审问、对峙和权力关系。
8. 鱼眼窥视：广角畸变或窥孔感，用于监控、梦境、惊悚、密闭空间心理压迫；慎用。
9. 遮挡后显露：镜头横移或主体移动，让人物/道具从墙、门、柱子、前景后出现，用于悬念和出场。
10. 穿越式运镜：镜头穿过门框、窗户、玻璃、车厢、人群缝隙进入新空间，用于转场和沉浸。
11. 由虚到实：画面从模糊逐渐清晰，用于揭示人物、苏醒、回忆结束、注意力建立。
12. 焦点切换：焦点从前景切到背景，或从道具切到人物，用于信息转移和关系反转。
13. 向上仰拍/上摇：镜头从低处向上看或上摇，用于人物压迫感、权威感、强势登场。
14. 向下俯拍/下摇：镜头向下扫到手部、脚步、道具或低处人物，用于弱势、发现线索、情绪坠落。
15. 横向左移：镜头向左平移，用于浏览空间、制造视差、揭示左侧人物或物件。
16. 横向右移：镜头向右平移，用于跟随行动、揭示右侧信息、建立横向关系。
17. 半环绕：围绕主体小幅弧形运动，用于关系变化、犹豫、压迫逐渐增强。
18. 快速全环绕：围绕主体快速 360 度运动，用于动作高点、强烈混乱、超现实心理冲击；避免普通对话。
19. 慢速弧形运镜：围绕主体缓慢弧移，用于克制情绪、浪漫、审视、电影感转折。
20. 垂直下移：机位下降或镜头下移，用于情绪低落、人物被压制、从脸落到手部动作。
21. 垂直上移：机位上升或镜头上移，用于觉醒、掌控感增强、从局部升到人物全貌。
22. 摇臂上抬：镜头从人物或地面升起，带出环境规模，用于段落开始、离别、命运感。
23. 摇臂下降：镜头从高处落到人物或道具，用于从环境进入人物处境、落点聚焦。
24. 平滑向内变焦：固定机位逐渐放大主体，用于心理逼近、压迫和注意力集中。
25. 平滑向外变焦：固定机位逐渐缩小主体，用于揭示处境、孤独感、关系距离。
26. 急速变焦：突然冲到主体或细节，用于震惊、发现、喜剧反应、反转提示；同段落少用。
27. 无人机高空俯掠：高空快速掠过地形或建筑，用于大场景建立和动作路线交代。
28. 无人机史诗揭露：从遮挡物后升起或越过边界揭示宏大场景，用于开场、转场、目标显现。
29. 无人机大范围环绕：围绕建筑、车辆、人群或地标大范围旋转，用于空间关系和规模感。
30. 上帝视角垂直俯拍：从正上方看人物或空间，用于布局、追踪路线、命运压迫和孤立感。
31. 穿越机俯冲：第一视角高速俯冲、穿越、贴近目标，用于动作、追逐、极强动势；真人短剧慎用。
32. 手持纪录片：轻微晃动、跟随呼吸感，用于真实、紧张、临场感、争执和追赶。
33. 快速甩镜：快速横摇或甩切到新主体，用于突然发现、道具反转、视线转移和节奏爆点。
34. 荷兰式倾斜：画面倾斜构图，用于不安、失衡、危险预感；不要长期保持。
35. 引导式跟拍：镜头倒退，人物向镜头走来，用于压迫、逼近、强势推进。
36. 跟随式跟拍：镜头从人物身后或侧后跟随，用于进入空间、逃离、追踪和行动连续性。
37. 平行侧跟：镜头与人物并排行进，用于横向运动、奔跑、争执同行、街道/走廊调度。
38. 第一人称视角行走：以角色视角进入空间或靠近对象，用于主观恐惧、探索、沉浸式代入。

## 运镜选择规则

- 情绪递进优先选：慢速推进、慢速后拉、平滑向内/向外变焦、慢速弧形运镜、焦点切换。
- 爆点反应优先选：`快速推进`、`急速变焦`、`快速甩镜`、`过肩镜头`，必要时可用 `固定大特写（静态例外）` 稳定读取眼神或道具。
- 空间揭示优先选：遮挡后显露、横向平移、摇臂上抬/下降、无人机揭露、上帝视角。
- 行动连续优先选：手持纪录片、引导式跟拍、跟随式跟拍、平行侧跟、穿越式运镜。
- 空镜和长对话承接优先选：`焦点切换` 从人物手部/文件切到空座位或货箱，`垂直下移` 从脸落到手部/道具，`垂直上移` 从证据升到人物侧脸，`慢速后拉` 或 `平滑向外变焦` 揭示压迫空间，`遮挡后显露` 用门框/货架/肩背带出出场人物，`由虚到实` 建立注意力，`横向左移/横向右移` 揭示门口、文件、样衣或站位变化。空镜必须承接台词、道具、空间压力或人物入场，不写无信息的漂亮空镜。
- 心理异常优先选：眩晕效应、鱼眼窥视、荷兰式倾斜、快速全环绕；只在剧本明确有失控、惊悚、眩晕、梦境或强心理冲击时使用。
- 高速/大范围镜头优先用于横版电影和 3D 动画；竖版短剧只在开场、追逐、反转或强钩子中短促使用。

## 动漫动作长镜头节奏

当视觉风格为 `2d-anime`，且用户要求打斗、追逐、超燃动作、一镜到底、长镜头或故事板动态分镜时，按以下节奏组织镜头。该规则优先服务动漫动作提示词，不用于普通对白段落。

- 节奏骨架：`建立方向 -> 主角加速 -> 第一次接触 -> 冲击爆点 -> 镜头被动作带走 -> 空间翻转 -> 新目标出现 -> 再次加速 -> 近身碰撞 -> 抛飞/坠落 -> 镜头穿过遮挡 -> 重新锁定主角 -> 更大范围追逐 -> 终极碰撞`。可按剧本缩短，但不要打乱因果顺序。
- 动作带镜头：镜头运动必须由角色冲刺、挥击、闪避、抛飞、坠落、碎片飞散或能量轨迹触发；提示词中写清“镜头被冲刺方向拖走”“顺着击退方向甩出”“穿过碎石后重新锁定@角色”。
- 动势接力：相邻动作段落必须有接力点，优先使用拳脚方向、武器弧线、身体飞行方向、碎石飞散方向、速度线、冲击波扩散、视线落点或新敌人出现方向；不得让下一段从无关方向突然开始。
- 远近交替：长动作段落避免一直近景。优先按 `远景/大全景建立空间 -> 中景/跟拍看冲刺 -> 近景/特写看碰撞 -> 半拍定格读姿态 -> 远景/后拉释放速度` 循环。
- 爆点半拍：重击、闪避、落地、反击前允许短促定格、背景压暗、速度线汇聚或角色剪影停顿；停顿后必须接快速推进、快速甩镜、穿越式运镜、跟随式跟拍、平行侧跟或快速全环绕释放动作。
- 遮挡重锁定：允许用烟尘、墙体、碎石、前景剪影、速度线、门框、玻璃裂纹或敌人肢体遮挡转场；遮挡后必须写清镜头重新锁定的主体、动作落点和运动方向。
- 空间翻转要可读：使用快速全环绕、穿越机俯冲、第一人称视角行走、跟随式跟拍或半环绕时，必须保留角色剪影、地平线/建筑边缘/透视线等方向锚点，避免只写旋转混乱。
- 动漫画面减法：高速段落优先用灰阶背景、粗线稿、剪影、速度线和1到2种高饱和动作色块突出动势；不要让粒子、光效、残影覆盖角色关键姿态。
- 一镜到底不等于不分段：可以在同一个镜头描述内写多个动作小节，但每个小节必须有动作起点、方向、遮挡或接力点、重新锁定点和爆点结果。

## AI光影技巧库（20种）

以下光影技巧用于生成 AI 视频提示词时选择。光影必须有来源、方向、色温和叙事目的；不要只写“电影级光影”。单镜头最多使用一种主光影风格，可叠加一个弱辅助光，例如“窗光主光 + 轻微轮廓光”。

1. 英雄混合色温光：暖色主光与冷色边缘光/环境光并存，用于人物高光时刻、强者登场、命运感。
2. 冷调月光夜景：偏蓝月光从侧后方或高处打入，用于夜晚、孤独、危险、秘密行动。
3. 蝴蝶光：主光从人物正前上方落下，在鼻下形成对称阴影，用于精致肖像、女主高光、商业感特写。
4. 伦勃朗光：侧前方主光在暗侧脸颊形成三角亮区，用于悬疑、权谋、压抑对话、人物复杂性。
5. 黄金时刻：日出后或日落前的柔暖低角度光，用于回忆、告白、释怀、温柔人物关系。
6. 蓝调时刻：日落后或日出前的冷蓝环境光，用于等待、悬念、孤独、城市夜色。
7. 体积光/神圣光束：光线穿过雾气、灰尘或烟尘形成可见光束，用于空间纵深、神秘感、舞台感。
8. 月光高光与皮肤反射：冷月光在皮肤、泪水、雨滴、金属上形成局部反光，用于真实夜景和细节质感。
9. 赛博霓虹双色光：洋红/青蓝等互补高饱和光从两侧分割人物，用于赛博、都市夜店、科技感、欲望场景。
10. 电视/屏幕光：冷色电子光从下方或正前方照亮脸部，用于手机、监控、电脑、秘密信息揭露。
11. 高调光：明亮、低阴影、干净均匀的光，用于广告、办公室、白天喜剧、轻松场景；不适合阴谋或恐怖。
12. 轮廓光：从背后或侧后方勾出人物边缘，用于主体分离、强势剪影、雨夜/走廊/舞台出场。
13. Gobo图案投影：窗格、百叶窗、树影、栏杆等图案投到人物或墙面，用于悬疑、禁锢、秘密和空间层次。
14. 底光：光从人物下方打上来，用于恐怖、反派、诡异告白或火光近脸；竖版真人慎用。
15. 动态烟花/彩色反射：多色闪烁光反射在脸部或环境上，用于庆典反差、浪漫、混乱、情绪爆发。
16. 书本光/大面积柔光：通过反射或扩散形成柔和包裹光，用于高质感人像、奢华、温柔、安静对话。
17. 负补光：在暗侧减少反射，保留深阴影，用于脸部塑形、压迫、秘密、真实电影感。
18. 浮雕式局部光：只照亮主体局部，背景接近全黑，用于人物被孤立、审判感、舞台式揭露。
19. 剪影光：主体变暗，背景明亮，人物以形体轮廓表达，用于离别、危险、身份隐藏、动作造型。
20. 双主体彩色剪影：两个人被不同颜色或方向的背光分开，用于对峙、黑色电影感、关系撕裂。

## 光影选择规则

- 先写光源动机：窗光、月光、屏幕、车灯、霓虹、火光、舞台灯、走廊顶灯、烟花、投影、雾中光束。
- 再写方向：正前上方、侧前方、侧后方、背后、下方、高处、画面左侧、画面右侧。
- 再写色温：暖色约 3000K，日光/干净白光约 5000K，冷蓝夜景可写“冷蓝月光/蓝调环境光”。不要在同一镜头里混入太多色温。
- 人像塑形优先选：蝴蝶光、伦勃朗光、轮廓光、书本光、负补光。
- 夜景氛围优先选：冷调月光夜景、蓝调时刻、月光高光与皮肤反射、屏幕光、霓虹双色光。
- 悬疑压迫优先选：伦勃朗光、Gobo图案投影、负补光、浮雕式局部光、剪影光。
- 反转爆点优先选：屏幕光、轮廓光、底光、动态彩色反射、双主体彩色剪影。
- 温柔/治愈优先选：黄金时刻、书本光、高调光、柔和窗光。
- 体积光只在有雾、尘、烟、雨汽、舞台烟雾或强逆光空间中使用；不要无来源滥用“丁达尔光”。

## 竖版短剧镜头

适用于 `vertical-short`。

### 景别

- 主力景别：中近景、近景、特写、大特写、反打、过肩、插入。
- 辅助景别：中景用于建立对话轴线；全景只用于站位、门口来人、关系压迫、人物进退。
- 竖屏短剧不得出现群像镜头，不得把多人同时作为镜头主体；多人场景优先只给有台词、被冲击、拿道具或推动下一步行动的角色面部近景/特写，其他人只能作为现场声音、前景肩背虚化、手部局部、单个路人近景/中近景、场景元素、关键道具特写或主角反应反打出现。
- 禁止用长时间远景替代冲突表达。
- 面部特写必须保留头顶、下巴和手部安全区。
- 双人关系镜头不得使用左右对称、平分画面的站位；优先用前后纵深、一坐一站、高低压迫、桌面/门框/窗框/道具阻隔、过肩遮挡或视线俯仰制造竖屏非对称压迫感。
- 两人交互时可以使用过肩镜头，或把其中一人作为虚化前景拍摄对面主要人物；画面主体必须是被观看、说话或反应的主要人物，不是前景人物。写清“前景人物肩背/微侧脸虚化、远背景虚化、焦点锁在@主要人物脸部/眼神/手部动作上”，避免前景抢焦。
- 两人单独对话都是面部近景时，摄像机位置不必写得过度复杂，只需绑定座位、肩后、门侧、中控台、桌沿等清楚锚点，并保证人物不是纯正脸看镜头。可写轻微侧角或微侧脸，视线落在对手、道具、窗外或地面。
- 两人对峙的脸部特写必须体现权力关系：强势方可用轻微仰拍和轻微侧角，弱势方可用轻微俯拍或略高机位，视线被压低或向上承接压力。不要把强弱双方都拍成同角度正脸平视。
- 竖屏真人人物特写不得默认正脸推进。写人物表演时优先写“@角色 微侧脸看向对手/文件/信封/门口/地面，眼睛不看镜头”，或“@角色 侧脸轮廓在前景遮挡后出现，视线落在画外对象”。除自拍视频、直播、监控口供或第一人称主观对视外，不写正脸看向镜头。

### 运镜

- 优先：`慢速推进`、`快速推进`、`过肩镜头`、`焦点切换`、`由虚到实`、`遮挡后显露`、`垂直上移`、`垂直下移`、`手持纪录片`、`快速甩镜`、`引导式跟拍`、`跟随式跟拍`、`第一人称视角行走`。稳定读口型、反应或道具时可少量使用 `固定镜头/固定特写（静态例外）`。
- 竖版视觉递进链运镜：按 `异常/目标先出现 -> 人物入场或靠近 -> 道具/局部引导注意力 -> 遮挡揭示人物反应 -> 关系定格或行动入口` 选择库项。异常/目标优先 `由虚到实`、`焦点切换`、`慢速后拉`、`平滑向外变焦`；入场/靠近优先 `遮挡后显露`、`跟随式跟拍`、`引导式跟拍`、`第一人称视角行走`、`横向左移/横向右移`；道具/局部优先 `垂直下移`、`垂直上移`、`微距变焦`、`焦点切换`；遮挡揭示优先 `遮挡后显露`、`由虚到实`、`穿越式运镜`、`过肩镜头`；关系定格优先 `过肩镜头`、`慢速推进`、`慢速后拉`、`平滑向外变焦` 或少量 `固定镜头/固定特写（静态例外）`。
- 视觉递进链必须写清信息落点：例如“焦点切换从前景珠帘落到床榻上的@男二”，“垂直上移从@女主手中灯笼升到她微侧脸”，“遮挡后显露让@女主从屏风后露出肩背并停在床榻前三步”。不要只写“氛围感运镜”。
- 爆点：`快速推进`、`急速变焦`、`快速甩镜`，或 `固定大特写（静态例外）`；爆点镜头应短、准、落在脸部反应或关键道具上。
- 对话：优先 `过肩镜头`、前景人物虚化看对面主体、说话者近景/特写、倾听者反应特写；可用 `慢速推进` 小幅靠近主体来强化台词压力。当前景人物入画时，必须写清前景虚化、背景虚化、焦点锁在主要人物上。
- 道具线索：`微距变焦`、`焦点切换`、`向下俯拍/下摇` 到手部/物件、`快速甩镜` 落到关键物。
- 空镜/出场/长对话承接：优先 `遮挡后显露` 让人物从门框、货架、肩背后出现；`由虚到实` 从布料、文件或门口虚焦切实；`垂直下移` 从说话者微侧脸落到手部/文件；`垂直上移` 从关键道具升到人物微侧脸；`焦点切换` 在道具、虚化前景和人物之间转移；`慢速后拉` 或 `平滑向外变焦` 揭示空间压力。空镜时长通常1到2秒，承接重台词或关键证据读数才可到3秒；组尾不得添加无信息空镜。
- 新人物入场：角色首次入镜或从画外进入场景时，必须把入场拍成可见过程，优先 `遮挡后显露`、`横向左移/横向右移`、`跟随式跟拍`、`引导式跟拍`、`第一人称视角行走`、`由虚到实` 或前景肩背遮挡反打。镜头描述必须写清起点、运动方向和落点，例如从走廊门框后露出侧脸并跨进门内侧，从画面右边缘走到桌前，或由画外脚步声接到门缝后显露人物。
- 谨慎：横向左移/右移只在揭示门口来人、道具位置或站位变化时使用；半环绕只做小幅度，不抢人物表演。
- 限制：无人机、超远距变焦、摇臂大升降、快速全环绕、穿越机俯冲通常不适合竖版日常对话；除非剧本是追逐、灾难、开场大场面或 3D/2D 风格化段落。
- 留白：只在台词、动作结果或关键道具信息之后需要停顿时使用 `慢速后拉`、`平滑向外变焦` 或 `固定特写（静态例外）`；竖屏短剧不得把普通环境空镜当作组尾留白。

### 镜头控制

- 单镜头只承载一个主要戏剧目标，但允许包含同一表演链里的1句短台词、1个轻动作和1个即时微反应。不要因为眨眼、低头、停顿、抿唇等微表情单独拆镜。
- 每个竖屏短剧镜头必须有剧情推进功能：人物开口、人物移动、视线转向新对象、关系压力变化、关键道具被使用或关键证据被确认。只说明“车通电但没点火”“屏幕亮着”“灯开着”“环境很安静”等状态的镜头必须删除，把这些信息并入本组 `【空间构图与人物位置】` 或相邻人物镜头。
- 镜间衔接必须继承上一镜结束时的人物场景锚点、姿态、手部落点、持物、道具落点和视线方向；摄像机锚点可以换，人物锚点不能无因变化。人物从蹲到站、站到坐、桌前到门口、拿起到放下、转身或离场时，必须用可见动作过程和运动方向连接。
- 竖屏短剧每组第一个镜头不得使用反打镜头作为起手；组首优先用当前说话者动作、手部动作、新落点、过肩关系、有效道具或入场动作承接剧情。
- 竖屏短剧每组结尾不得添加没有意义的人物反应、普通停顿、环境空镜或余韵镜头；组尾必须停在台词、动作结果、关键道具落点或下一步行动入口上。
- 轻微心理/表情提示不得扩写成独立镜头；“环顾四周、面露嫌弃、似笑非笑、皱眉、冷笑、看了一眼、停顿半拍”等应并入紧邻的台词镜头、动作镜头或同框表演，除非它们直接触发信息反转、关键道具落点、人物换位或权力关系变化。
- 每 2 到 4 个镜头必须有信息推进；竖屏短剧不能连续堆普通反应、普通停顿或无信息空镜。
- 爆点台词后必须给被冲击者反应；普通反应可合并在同一镜头或同框表演中，一级爆点才安排独立反应镜头。
- 对话优先使用连续镜头或双人关系镜头承载短句往返；只有在台词重音、权力关系变化、关键道具出现、沉默停顿或情绪爆点时切入特写/反打。
- 不机械套用“过肩/中景 -> 说话者近景/特写 -> 倾听者反应特写 -> 回中景”循环；每2到4句普通对白可共用一个稳定关系镜头。
- 竖版运镜要保留人物脸部、手部、道具的安全区；避免大幅横移导致主体出画。

### 焦距

- 默认按全画幅等效焦段理解；提示词可写“标准镜头/中焦/中长焦/微距”，不必每镜都硬写数字。
- 主力焦段：35mm 到 50mm。竖版短剧的大多数中近景、近景、过肩、反打、普通反应和双人关系镜头优先用35mm、40mm、50mm或“标准镜头/中焦”，保证脸部自然、手部动作和道具仍在画面内。
- 24mm 到 35mm：只用于 `引导式跟拍`、`跟随式跟拍`、`平行侧跟`、`第一人称视角行走`、狭窄空间、门口来人或需要明显临场压迫的镜头；人物脸部必须靠画面中心，避免边缘拉伸和大鼻子畸变。
- 50mm 到 65mm：用于说话者近景、倾听者反应、轻微空间压缩和情绪聚焦，是竖版普通对白的安全焦段。
- 70mm：只用于情绪明显压近、权力关系变化、关键台词近景或短促反应压迫；不得作为普通对白反打的默认焦段。
- 85mm 到 100mm：只用于爆点特写、眼神特写、道具插入、手部动作、关键表情和身份反转；持续时间要短，避免裁掉下巴、手部或关键道具。
- 同一组内如无明确爆点，85mm最多出现1次；连续普通对白不得连续使用70mm以上焦段，避免竖屏空间被压得过窄、人物关系消失。
- 100mm 以上长焦/超长焦：竖版短剧慎用，只在偷窥、远处观察、强压缩孤立感或特殊风格化镜头中使用，不作为常规对话焦段。
- 道具插入：优先微距、50mm 到 85mm 中长焦浅景深；必须写清道具与人物动作关系，避免只拍漂亮虚化。
- 压迫感：优先用 35mm 到 50mm 近距离快速推进，或 50mm 到 70mm 轻压缩空间；不要默认用超长焦把竖屏画面压得过窄。
- 爆点特写：可用 70mm 到 100mm 中长焦配合 `快速推进`、`急速变焦`，或使用 `固定大特写（静态例外）`；快速推进时从标准镜头进入近景，比从超广角硬推更稳定。

### 光学特效

- 每个镜头最多使用一种主光影技巧；竖版短剧优先让光影强化脸部反应，而不是展示环境。
- 日常对话优先：柔和窗光、书本光/大面积柔光、轻伦勃朗光、轮廓光、眼神光、浅景深背景虚化。
- 夜间/秘密信息优先：冷调月光夜景、手机/电视屏幕冷光、蓝调时刻、月光高光与皮肤反射。
- 爆点/反转优先：屏幕光照亮脸、快速出现的轮廓光、Gobo图案落到脸上或墙面、短促底光；底光只用于恐惧、反派或火光近脸。
- 禁止滥用：满屏光斑、无来源强逆光、过量丁达尔、廉价柔焦、奇幻粒子、多色霓虹乱闪。
- 慎用：体积光、烟花彩色反射、双主体彩色剪影、浮雕式局部光；只有场景本身提供雾、烟、舞台、庆典、审判感或强对峙时才使用。

### 3D动画画面特效

- 3D动画可以使用克制的非剧情性画面特效：角色轮廓辉光、镜头辉光、微尘粒子、动作拖尾、冲击瞬间的轻微画面震动、布料和发丝二级运动、浅景深散景、材质高光闪动。
- 这些特效只能强化动作、情绪或空间层次，不能新增魔法能力、爆炸、武器、灾难、伤害结果或剧本没有的剧情事件。
- 对话镜头优先保证口型、眼神和角色绑定表演清晰；特效不得遮挡脸部、手部、道具和关键表情。

## 横版短剧镜头

适用于 `horizontal-short`。

### 景别

- 主力景别：中景、中近景、近景、双人镜头、过肩、反打、插入。
- 开场可用短暂全景或中全景建立空间，但不能拖慢钩子。
- 横版短剧允许更多双人同框，但必须能看出谁占上风、谁后退、谁压迫。
- 两人交互优先使用过肩镜头或前景人物虚化的反打关系镜头；前景只能作为遮挡和关系提示，焦点必须锁在对面主要人物脸部、眼神或手部动作上，远背景保持虚化或弱化。
- 两人对峙脸部特写要用机位区分强弱：强势方轻微仰拍并保持轻微侧角；弱势方轻微俯拍或略高机位，不使用双方同角度平视正脸来处理压迫关系。

### 运镜

- 关系建立：优先用双人同框、过肩镜头、前景人物虚化的反打、横向左移/右移、半环绕、慢速弧形运镜，展示谁占上风、谁被逼退、谁控制空间。
- 信息揭示：优先用遮挡后显露、由虚到实、焦点切换、平滑向内/向外变焦，把观众注意力从人物转到道具、手机、门口来人或隐藏证据。
- 爆点反应：从双人关系镜头切入反应特写，或用快速推进、急速变焦、快速甩镜落到脸部/道具；爆点后必须回到被冲击者反应。
- 行动跟拍：走廊、街边、办公室移动戏可用引导式跟拍、跟随式跟拍、平行侧跟、稳定手持；运动要短促，不要变成长片式铺陈。
- 空间揭示：可短暂使用摇臂下降、摇臂上抬、穿越式运镜或横向平移，但落点必须是人物冲突或剧情线索。
- 谨慎：眩晕效应、鱼眼窥视、荷兰式倾斜只用于明确心理失衡；快速全环绕只适合强动作、醉酒、混乱或高度风格化段落。
- 限制：无人机高空俯掠、无人机史诗揭露、无人机大范围环绕、穿越机俯冲只用于开场钩子、转场、追逐或大空间交代，不用于普通室内对话。

### 镜头控制

- 保留短剧节奏：钩子快、反应快、反转清晰。
- 横向环境只服务剧情，不做无意义铺陈。
- 对话可用双人同框承载一部分张力；关键台词后必须有明确反应，但普通反应可保留在同一镜头内，一级爆点再拆成反打或独立反应特写。
- 镜头之间注意轴线方向和视线方向，避免横屏对话跳轴。
- 一组横版短剧对话通常控制在“关系镜头 -> 说话者近景 -> 倾听者反应 -> 道具/信息插入”四类镜头内循环。
- 运镜变化要跟台词重音或动作触发点绑定；不要每个镜头都移动。

### 焦距

- 默认按全画幅等效焦段理解；横版短剧比竖版可容纳更多空间，但仍以人物反应为主。
- 主力焦段：28mm 到 85mm。既保留横向关系，又不让人物变小。
- 24mm 到 35mm：用于开场空间、门口来人、多人站位、走廊/街道跟拍；人物靠近画面边缘时避免脸部畸变。
- 35mm 到 50mm：用于双人同框、对峙、办公桌/餐桌关系镜头，是横版短剧最安全的关系焦段。
- 50mm 到 85mm：用于反打、倾听者反应、轻压缩空间和爆点近景。
- 85mm 到 135mm：用于身份揭露、远处偷看、强压迫反应或背景虚化；不要长时间用于普通对话。
- 道具插入：微距、50mm 到 100mm 中长焦浅景深；道具必须推动反转或关系变化。
- 快速推进/急速变焦：优先从 35mm 到 50mm 的关系镜头压到 70mm 到 85mm 的近景，不要从超广角硬推到畸变脸。

### 光学特效

- 日常对白：柔和窗光、书本光/大面积柔光、高调光、轻轮廓光；画面干净，表情优先。
- 权力对峙：伦勃朗光、负补光、侧后方轮廓光、混合色温，用明暗和冷暖区分强弱双方。
- 秘密/反转：手机/电视屏幕冷光、门缝光、Gobo图案投影、蓝调时刻，用光源把线索导向人物脸或道具。
- 都市夜景：可用赛博霓虹双色光、玻璃反射、月光高光与皮肤反射，但色彩不要盖过台词反应。
- 庆典/情绪爆发：可短促使用烟花彩色反射或双主体彩色剪影，必须有现场动机。
- 不要让光效抢走人物反应。光学效果必须有场景来源，并且不能破坏短剧节奏。

## 横版电影镜头

适用于 `horizontal-film`。

### 景别

- 可按段落从远景/全景进入中景、中近景、近景、特写。
- 建立镜头要交代空间关系、光源方向和人物处境，不只是展示环境。
- 特写用于人物决定、情绪临界点、关键道具和信息反转。

### 运镜

- 可用完整运镜库，但必须按段落目的选择，不要为了炫技堆镜头。
- 建立段落：超远距变焦、无人机高空俯掠、无人机史诗揭露、无人机大范围环绕、上帝视角垂直俯拍、摇臂上抬/下降，用于交代空间、路线、环境压力。
- 进入段落：遮挡后显露、穿越式运镜、横向平移、跟随式跟拍、第一人称视角行走，用于带观众进入房间、走廊、车内、人群或危险现场。
- 情绪段落：慢速推进、慢速后拉、眩晕效应、焦点切换、慢速弧形运镜、垂直上移/下移、平滑向内/向外变焦、荷兰式倾斜，用于心理变化和关系转折。
- 对话段落：过肩镜头、前景人物虚化的反打、半环绕、慢速推进、焦点切换、反打特写；长镜头内必须有站位、视线、距离或权力关系变化。前景人物入画时，前景和远背景都要虚化，焦点锁在对面主要人物上。
- 行动段落：快速推进、快速全环绕、穿越机俯冲、手持纪录片、快速甩镜、引导式跟拍、跟随式跟拍、平行侧跟，用于追逐、逃离、冲突升级和身体动作。
- 余韵段落：慢速后拉、平滑向外变焦、剪影静止、环境空镜、道具特写，用于沉默、后果和段落收束。
- 电影镜头允许更长的运动，但镜头内部必须有信息变化。
- 重要段落可用镜头组：建立空间 -> 角色移动 -> 对峙关系 -> 反应特写 -> 道具/环境余韵。

### 镜头控制

- 允许节奏呼吸，但不能添加剧本没有的宏大奇观。
- 光影、焦距、站位和人物距离共同表达关系变化。
- 对话不必每句都切；关键台词、沉默、转身、道具声响和眼神变化是切镜触发点。
- 反打镜头必须保持光位、轴线和视线方向连续。
- 两人对峙的脸部特写必须通过机位表达强弱：强势方轻微仰拍并保持轻微侧角，弱势方轻微俯拍或略高机位，保持眼神方向和轴线连续。
- 每个段落先确定“空间目标”和“情绪终点”，再选择运镜；镜头运动必须把人物带向新的关系位置或新信息。
- 长镜头内部至少包含一次信息变化：人物站位改变、视线改变、道具出现、光线变化、空间层级揭示或情绪反转。

### 导演分镜与转场

横版电影镜头之间必须有接力点，不得只因为画面好看而切换。每次切镜前先判断触发点：动作、视线、台词重音、沉默、道具声响、光线变化、人物距离变化或情绪转折。

优先使用以下转场逻辑：

- 动作中断转场：人物动作未完成时切走，用于悬念、危险、犹豫和节奏加速。必须写清动作进行到哪个阶段被切断，例如手刚碰到门把、脚刚踩出门槛、手机刚离开掌心。
- 视线转场：上一镜人物看向某物或某人，下一镜切到被观看对象、对象反应或更大的空间后果。必须写清上一镜视线落点和下一镜承接主体。
- 道具转场：上一镜道具落点、手部动作或道具声音，下一镜接道具后果或人物反应。适合信封、手机、钥匙、血迹、机器按钮、照片和文件。
- 声画转场：台词、呼吸、脚步、机械声、环境声延续到下一镜，画面主动换主体。用于跨组承接长台词、隐藏说话者或制造空间外压力。
- 尺度转场：极近特写切大全景，或大全景切眼神/手部特写，用于史诗感、压迫感和心理落差。尺度变化必须服务信息变化，不得只做视觉冲击。
- 形状/光色转场：相似轮廓、相同色温、同方向运动连接两个空间。必须服务剧情信息，例如屏幕冷光切到病房冷白灯、圆形机器镜头切到人物瞳孔。

禁止连续两镜只是同一人物、同一景别、同一机位的重复反打。下一镜应主动错开主体、景别、机位、焦点或信息落点。

### 机位权力关系

横版电影的机位角度必须服务人物关系、空间压迫或心理距离：

- 低角度仰拍：权威、压迫、神性、机器感、人物掌控局面。
- 高角度俯拍：弱势、被审视、命运压迫、人物被空间吞没。
- 平视正面：审判、对峙、无法逃避的凝视。
- 轻微侧角：隐藏动机、保留人物复杂性，不把人物解释干净，同时避免纯正脸看镜头。
- 过肩反打：关系压力、视线归属和谈判位置。
- 前景遮挡：秘密、窥视、距离、阻隔和被监视感。
- 极近特写：眼睛、手、嘴、道具，用于情绪临界点和信息确认。
- 大远景/全景：人物与世界的关系，不用于承载细腻表演。

强势方和弱势方不得使用完全相同的平视正脸反打。必须通过机位高度、焦距、遮挡、光位或人物在画面中的占比体现关系差异。

### 段落情绪曲线

横版电影段落优先按以下曲线组织镜头：

```text
建立处境 -> 发现异常 -> 人物反应 -> 世界/关系揭示 -> 行动断点或余韵
```

- 建立处境：使用远景、全景、中景、遮挡后显露、摇臂下降或跟随式跟拍，交代人物被什么空间关系包围。
- 发现异常：使用焦点切换、向下俯拍/下摇、微距变焦或道具插入，把观众注意力转向新信息。
- 人物反应：使用近景、特写、过肩镜头、慢速推进或固定特写（静态例外），读取眼神、呼吸、手指和下颌变化。
- 世界/关系揭示：使用慢速后拉、平滑向外变焦、摇臂上抬、上帝视角垂直俯拍或大全景，揭示人物无法控制的后果。
- 行动断点或余韵：用动作中断、慢速后拉、环境空镜、道具声响、光影移动或人物离开画面收束段落。

### 焦距

- 默认按全画幅等效焦段理解；横版电影可使用更宽的焦段范围，但同一段落要有焦段逻辑。
- 18mm 到 24mm：用于大空间、建筑、荒野、上帝视角、主观压迫和强透视；人物近脸慎用。
- 24mm 到 35mm：用于环境与人物关系、手持压迫、跟拍、多人调度和空间进入。
- 35mm 到 50mm：用于自然视角、双人关系、中景调度和克制对话。
- 50mm 到 85mm：用于人物近景、反打、情绪聚焦和轻度空间压缩。
- 85mm 到 135mm：用于对峙压缩、孤立、偷窥、远处观察和背景剥离。
- 135mm 以上：只用于极强偷窥感、远距离追踪、命运式孤立或风格化压缩，不作为常规对话焦段。
- 关键物品：微距、50mm 到 100mm 中长焦浅景深；物品特写后要接人物反应或动作后果。
- 主观压迫：按语气选择广角近距离低机位，或长焦压缩空间；不要两种压迫方式同时堆叠。

### 光学特效

- 光学效果必须有动机：窗光、车灯、霓虹、投影、手机屏幕、火光、雨夜反光、走廊顶灯、月光、烟尘、舞台灯、烟花。
- 人像塑形：蝴蝶光用于精致/高光肖像；伦勃朗光用于复杂人物；书本光用于柔和高级感；负补光用于压迫和秘密；轮廓光用于主体分离。
- 夜景系统：冷调月光夜景、蓝调时刻、月光皮肤高光、屏幕光和雨夜反光可组合成统一夜景，但同镜头不超过一种主色温对比。
- 风格化空间：赛博霓虹双色光、Gobo图案投影、体积光、浮雕式局部光、剪影光、双主体彩色剪影可用于强类型段落；必须先交代光源来源。
- 情绪转折：黄金时刻用于释怀/告白；底光用于恐惧/反派/火光；烟花彩色反射用于庆典与内心反差；高调光用于洁净、讽刺或表面平静。
- 电影段落允许更强的明暗反差，但必须保留人物眼神、轮廓和动作读数；不要让黑场或强逆光吞掉剧情信息。
- 同一镜头最多一个主光影风格；不要用特效替代人物表演。

## 视觉风格适配

- 真人/仿真人：镜头和光效必须像真实剧组可拍摄，保留眼神光、口型和皮肤纹理。优先使用慢速推进、过肩、焦点切换、手持跟拍、横移、半环绕；谨慎使用鱼眼、眩晕、快速全环绕和穿越机。
- 3D动画：镜头可更稳定和可控，可适度使用穿越式运镜、全环绕、无人机、穿越机、上帝视角；角色绑定表演必须清晰，光效和非剧情性画面特效可以风格化但要克制。
- 日漫2D：镜头可以转译为分镜格、定格、速度线、背景压暗、眼部大特写、漫画留白、快速甩镜和焦点切换，但仍要遵守所选成片路线。
