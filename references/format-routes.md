# 成片路线规则

## 竖版短剧：vertical-short

用于手机端9:16短剧。

- 构图必须写明 `9:16竖屏构图` 或 `超纵深垂直构图`。
- 人脸表演优先落在画面上中部或中轴偏上区域。
- 眼睛接近上三分线；胸口以上特写保留头顶和下巴安全距离。
- 手部动作和关键道具放在胸前或画面下中部安全区。
- 景别优先：中近景、近景、特写、大特写、过肩、反打、插入。
- 全景只用于交代站位、权力关系、人物进退和空间阻隔。
- 禁止群像镜头；多人场景优先只给有台词、被冲击、拿道具或推动行动的角色面部近景/特写，其余人物用前排单个路人、手部/肩背局部、虚化遮挡、现场声音、场景元素、关键道具特写或主角反应反打替代。
- 相邻镜头必须继承人物场景锚点、身体姿态、手部落点、持物、道具落点和视线方向；摄像机锚点可以变化，人物锚点不能无因变化，人物移动必须写出可见过程。
- 运镜优先从 `camera-rules.md` 的 `AI运镜手法库` 调用原名：`慢速推进`、`快速推进`、`快速甩镜`、`手持纪录片`、`垂直上移`、`垂直下移`、`引导式跟拍`、`跟随式跟拍`、`过肩镜头`、`焦点切换`。需要稳定读口型、反应或道具时，才可写 `固定镜头/固定特写（静态例外）`。
- 开场3秒必须有冲突信号：异常结果、关键道具、震惊反应、压迫站位或高压台词。
- 单镜头只传递一个主要信息点。
- 镜头规则必须走竖版镜头逻辑：竖向安全区、脸部上中部、纵向推拉、垂直升降、近距离反应特写。

### 竖版短剧视觉递进逻辑

竖版短剧默认按“先给异常/目标，再让人物靠近，再用局部或道具引导，再通过遮挡揭示反应，最后锁定人物关系”的逻辑组织镜头。该逻辑适用于古风、现代、家庭、职场、悬疑、甜宠和3D动画等竖屏短剧，不局限于珠帘、灯笼或卧室场景。

```text
异常/目标先出现 -> 人物入场或靠近 -> 道具/局部引导注意力 -> 遮挡揭示人物反应 -> 关系定格或行动入口
```

- 异常/目标先出现：开场优先给床榻醉卧、门口背影、桌上证据、空座位、亮起的手机、被打翻的物件、压迫站位或画外声音来源。它必须是剧情钩子，不是普通环境展示。
- 人物入场或靠近：新人物必须从门框、走廊、屏风、车门、桌侧、画面边缘、虚化肩背或画外脚步声方向进入；写清移动方向和落点，不能直接放到最终站位。
- 道具/局部引导注意力：用手、脚步、信封、手机、灯笼、杯子、钥匙、衣角、门把、珠帘、纱帘等局部把观众视线带向下一信息点。道具必须和人物动作或剧情信息有关。
- 遮挡揭示人物反应：门框、帘子、屏风、前景花、肩背、玻璃、文件、货架等遮挡可以隐藏身份或延迟情绪；遮挡打开后必须揭示人物脸部反应、手部动作、道具落点或关系变化。
- 关系定格或行动入口：段落结尾要把谁看见谁、谁控制空间、谁被发现、谁准备行动写清楚；停在台词、动作落点、关键道具落点或下一步行动入口上，不补无意义氛围空镜。
- 光源递进可作为视觉逻辑：灯笼、手机、窗光、车灯、烛火、屏幕、走廊灯等移动或局部光源，可以按“照亮脚步/手部 -> 照亮道具 -> 照亮脸部 -> 照亮关系对象”的顺序推动信息。
- 遮挡层次要服务叙事：前景虚化负责偷窥或隐藏，中景遮挡负责人物入场或身份保留，背景目标负责剧情钩子。不要让花瓣、粒子、珠帘、货架或肩背遮住口型、眼神、手部和关键道具。
- 若用户给出参考分镜，优先吸收这个递进关系，而不是照抄每秒时长；短句台词、轻微表情和普通动作仍按竖版短剧节奏重新校准。

## 横版短剧：horizontal-short

用于16:9横屏短剧。它不是电影分镜，仍然服务短剧爽点、反应镜头和快速推进，只是利用横屏承载更多横向站位和环境关系。

- 构图必须写明 `16:9横版短剧构图`。
- 节奏仍按短剧逻辑推进：钩子、空间关系、起势动作、反应特写、台词释放、对方反应、反转或道具落点、余韵。
- 允许更多双人同框、横向对峙、前景遮挡、门窗框架、桌面分割、走廊纵深。
- 景别优先：中景、中近景、近景、双人镜头、过肩、反打、插入；开场可使用短暂全景建立空间。
- 不要把横屏变成慢节奏电影铺陈。每3到5个镜头仍需有信息推进。
- 爆点台词后仍必须给被冲击者反应。
- 运镜可使用滑轨平移、推拉、横向跟拍、过肩反打、稳定手持；避免为展示环境牺牲人物脸部可读性。
- 镜头规则必须走横版短剧逻辑：横向站位、前后遮挡、双人关系线、横向运镜，但节奏仍按短剧推进。

## 横版电影：horizontal-film

用于16:9电影分镜、预告片、长片片段或电影感叙事。重点是镜头组、空间调度、光影、节奏呼吸和人物在环境中的关系。

- 构图必须写明 `16:9横版电影构图`。
- 允许更完整的建立镜头、移动镜头和环境段落，但每个镜头仍要服务剧情信息。
- 景别可从远景、全景、中景逐步进入近景和特写，形成段落层次。
- 运镜可使用推轨、拉轨、横移、摇镜、升降、环绕、长焦压缩、广角纵深。
- 对话不必每句都切，但关键台词、信息反转和人物关系变化必须有明确反应镜头。
- 光影必须有动机：窗光、台灯、走廊灯、车灯、手机屏幕、病房冷白灯、办公室顶灯等。
- 允许更长的余韵镜头，但画面必须有可见变化：呼吸、泪光、手指松开、道具声响、人物远去、光影移动。
- 不要为了电影感添加剧本没有的宏大场面、武器、爆炸或奇观特效。
- 镜头规则必须走横版电影逻辑：镜头组、空间段落、焦距变化、光影连续性、动机化光学效果。

### 横版电影导演分镜原则

横版电影分镜不得只是镜头列表。每个镜头必须回答一个导演问题：观众在这一镜获得了什么新信息，人物关系发生了什么变化，下一镜为什么必须接在这里。

优先使用“信息台阶”组织段落：

```text
建立处境 -> 发现异常 -> 人物反应 -> 世界/关系揭示 -> 行动断点或余韵
```

- 单镜头只承担一个主要信息，但镜头内部可以包含动作起点、表演变化和动作落点。
- 不要在同一镜头里同时完成环境展示、人物情绪、道具揭示和关系反转。
- 建立镜头不是空泛展示环境，必须交代人物被什么空间、制度、距离、光线或道具关系包围。
- 反应镜头不是表情补丁，必须显示人物接收信息后的可见变化：眼神落点、呼吸、肩颈、手指、下颌、站姿或持物状态。
- 奇观镜头只在剧本需要世界观揭示、权力压迫或空间后果时使用；否则优先回到人物、道具和关系。

## 路线转换

竖版转横版短剧时：

- 保留短剧节奏和反应优先。
- 把竖屏安全区改为横向站位、前后层次和人物关系线。
- 增加环境信息，但只增加与剧情有关的部分。

横版短剧转横版电影时：

- 放慢个别建立镜头和余韵镜头。
- 加强光影动机、空间调度和镜头组连续性。
- 减少过密台词切镜，但保留爆点反应。

横版电影转横版短剧时：

- 缩短建立镜头。
- 增加反应特写和道具插入。
- 每3到5个镜头强化一次信息推进。
