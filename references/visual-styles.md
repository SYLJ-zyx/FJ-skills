# 视觉风格规则

## 真人 / 仿真人：live-action

用于真人、仿真人、实拍、真人短剧、真人电视剧质感。

- 渲染锁定可写：`电影级真人电视剧实拍质感`、`高真实感仿真人短剧质感`、`真人电影级镜头质感`。
- 光源必须有现实动机：窗光、台灯、走廊灯、车灯、手机屏幕光、病房灯、办公室顶灯。
- 面部保留真实皮肤纹理，避免过度磨皮、瓷娃娃皮肤、无来源发光和AI柔焦。
- 表演核心是微表情和微动作：眼睑、瞳孔、嘴唇、下颌、呼吸、喉结、手指、肩颈、站姿和道具互动。
- 表演描述只写可拍摄的生理指标和身体动作，不写文学化修辞或抽象心理。把“心如刀割、怒火中烧、杀气腾腾、绝望如深渊、柔情似水”等表达改成胸口起伏、下颌咬紧、眼神固定、手指颤抖、肩膀收紧、嘴角变化等字面画面。
- 镜头语言使用真实镜头词：标准镜头、35mm、50mm、85mm、中长焦、长焦、微距、浅景深。
- 不使用剧本没有的魔法光、满屏粒子、夸张特效、现代军械或奇观动作。

## 3D动画：3d-animation

用于3D、三维、CG、游戏电影感、三维动画电影质感。

- 渲染锁定可写：`高质量3D动画电影质感`、`写实CG短剧质感`、`风格化三维角色动画质感`。
- 即使风格化，也要保留戏剧逻辑：站位、视线、反应、台词节奏和镜头推进必须成立。
- 把真人微表情转成角色绑定可读动作：眉弓压缩、眼睑弧线变化、瞳孔聚焦、口型变形、下颌开合、肩部压缩、手指蜷缩。
- 可以加入动画二级运动和非剧情性画面特效，但必须克制：发丝跟随、衣料摆动、轻微动作拖尾、面部绑定形变、身体重心变化、角色轮廓辉光、镜头辉光、微尘粒子、冲击瞬间的轻微画面震动、材质高光闪动。
- 9:16时注意角色头顶安全和脸部绑定可读；16:9时强调角色剪影、前后景分离和镜头路径。
- 不要每个镜头都堆粒子、辉光或强风格化特效；可在爆点、转身、起身、离场、道具落点和强反应镜头中使用少量非剧情性特效。魔法、爆炸、能量攻击、灾难等会改变剧情事实的特效，只有剧本明确需要时才写。

### 3D国漫精细建模短剧

用于用户要求3D国漫、古风国漫、国漫建模、精细建模、3D动漫短剧、3D国风短片、角色建模精细、电影氛围感光影时。该规则属于 `3d-animation`，不属于2D日漫。

- 渲染锁定优先写：`3D国漫精细建模质感`、`高质量3D国风动画短剧质感`、`古风国漫角色精细建模质感`。不要只写“国漫感很好”。
- 角色材质要具体但不扩写剧本外貌：可写皮肤次表面散射、眼球高光、发丝分束、布料法线纹理、金属/玉石/木质/纱帘材质高光；人物身份、服饰颜色和发型不得凭空新增。
- 古风场景要绑定真实道具和光源：烛火、灯笼、月光、窗棂、珠帘、纱帘、床榻、屏风、香炉、花瓣、木柱、青砖、帷幔；每个光效必须来自画面内或空间中可解释的光源。
- 竖屏短剧构图要保证脸部绑定可读：人物脸部优先落在画面上中部，手、灯笼、珠帘、酒杯、信物等关键道具落在画面中下部；头顶留安全空间，避免珠帘或前景花瓣遮住口型和眼神。
- 表演使用3D绑定语言：眉弓压缩、眼睑弧线变化、瞳孔聚焦、口型开合、下颌收紧、肩颈微缩、手指蜷缩、持物手腕转动、身体重心前移或后撤。
- 氛围特效必须克制：可写暖黄烛光、灯笼局部照明、浅景深散景、微尘、花瓣虚化前景、纱帘半透明遮挡、材质高光闪动、发丝和衣料二级运动；不得写无来源魔法光、满屏粒子、能量波或改变剧情事实的奇观效果。
- 窥视、遮挡和入场镜头可优先使用珠帘、纱帘、屏风、门框、灯笼杆、花枝作为前景遮挡；遮挡后必须重新锁定人物脸部、手部道具或床榻上的关键人物，不能只展示漂亮环境。
- 醉酒、昏睡、关切、怀疑等状态必须转为可见动作：呼吸起伏、头偏向枕侧、手指松垂、眼睑半合、女主脚步放慢、眉心收紧、侧头看向床榻、灯笼抬高或压低。

## 日漫2D：2d-anime

用于2D、二维、日漫、日式动画、番剧感、赛璐璐、漫画动态分镜。

- 渲染锁定可写：`高质量日漫2D动画质感`、`赛璐璐日式动画分镜质感`、`日漫番剧级动态分镜质感`。
- 情绪通过可绘制符号表达：眼睑线颤动、瞳孔缩小、嘴线压紧、下颌轮廓收紧、手指弯曲、肩线下沉、发丝轻晃。
- 可使用日漫镜头语言：定格特写、反应停顿、插入分镜、速度线、背景压暗、眼部大特写、漫画式留白、前景剪影。
- 单镜头以一个关键姿态或一个动作转折为主，避免把完整连续动作塞进一镜。
- 竖版短剧时，把画面当成竖向漫画格：脸部上中部，手和道具下中部，纵深明确。
- 横版短剧时，保留短剧节奏，用横向对峙和反打制造爽点。
- 横版电影时，允许更长的定格、环境铺垫、人物背影和光影留白。
- 不写泛泛的“很日漫”“很好看”，要写线条、色块、构图、停顿和动作。

### 动漫一镜到底打斗节奏

用于用户要求动漫、2D、日漫、漫画动态分镜、故事板、超燃打斗、一镜到底、长镜头追逐或高速动作段落时。核心不是单纯加快镜头，而是让打斗动作拖动镜头移动：

- 动作先于镜头：角色冲刺、被击飞、翻滚、穿过遮挡或攻击落点先发生，镜头再被动作方向带走；不要写随机飞行镜头。
- 动势接力：上一动作的拳脚方向、武器轨迹、身体抛飞、碎片飞散、速度线、能量轨迹或视线落点，必须成为下一段镜头运动方向或新目标出现方向。
- 远近呼吸：高速动作段落按 `远景交代空间 -> 中景追随冲刺 -> 近景/特写表现碰撞 -> 爆点前半拍可读姿态 -> 远景释放速度` 交替，不要全程贴脸乱打。
- 遮挡重锁定：镜头穿过烟尘、碎石、门框、墙体、前景剪影或速度线遮挡后，必须重新显露并锁定主角、敌人或关键动作落点。
- 爆点半拍：命中前保留一个可读关键姿态或剪影停顿，命中后立刻用冲击波、速度线、画面震动、角色抛飞或镜头甩出释放能量。
- 减色保清晰：背景和人物可用灰阶、剪影或粗线稿，动作轨迹用1到2种高饱和强调色；不要让满屏特效遮挡角色轮廓。
- 故事板风格可写粗线、透视辅助线、速度线、手绘爆点线、色块残影和低帧率动势；成片日漫风格则写赛璐璐阴影、动态背景线、残影和关键帧姿态。
- 禁止只写“一镜到底打斗很流畅”；必须写清动作路径、镜头被哪个动作带走、穿过什么遮挡、在哪里重新锁定、爆点前后如何加速。

## 共同规则

- 所有人物用 `@角色名` 标记。
- 视频提示词不扩写人物外貌；人物以剧本角色名和 `@角色名` 标记为准。
- 默认无BGM、无字幕，只保留现场音效或画面内可听见的动作声。
- 视觉风格服务剧情，不替代表演、台词和镜头逻辑。
