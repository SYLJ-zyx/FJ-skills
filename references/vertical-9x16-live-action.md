# 竖版9:16真人短剧规则

本文件浓缩自用户提供的《真人5.0_短剧竖屏增强版.md》，用于 `vertical-short + live-action` 路线。

## 核心原则

生成面向手机端9:16播放的真人/仿真人短剧分镜提示词。镜头必须符合强钩子、强冲突、快节奏、有效反应和情绪爆点递进。人物表演必须通过可拍摄的面部肌肉、呼吸、眼神、唇部、下颌、手指等微表情与微动作外化。所有镜头必须服务台词、动作、关系变化或关键道具信息，不重要的环境描写和无因果反应必须删除。

竖屏短剧不得把单纯场景状态拆成独立镜头。车辆通电未点火、仪表盘亮起、房间灯亮、门牌、走廊、桌面摆设等环境或设备状态，如果不触发人物动作、不承接台词、不改变关系、不揭示关键证据，只能写进 `【空间构图与人物位置】`，不得单独生成空镜或插入镜头。

## 开场钩子

每段分镜开头3秒内必须出现冲突信号：

- 异常结果。
- 危险或关键道具。
- 人物震惊反应。
- 压迫性站位。
- 一句高压台词。

如果原剧本已经有强冲突，可直接用人物近景、对峙特写、关键动作插入或反应特写起幅，再用中景或全景补空间关系。不得用无人物动作、无台词承接、无关键证据的环境状态镜头作为开场钩子。

## 短剧节奏链

重要情节按以下顺序组织：

`钩子画面 -> 空间关系 -> 起势动作 -> 反应特写 -> 台词释放 -> 对方反应 -> 信息反转/道具落点 -> 余韵留白`

复仇、打脸、身份揭露、误会反转、告白、背叛、死亡威胁等爽点按以下顺序：

`被压迫者反应 -> 压迫者得意/逼近 -> 关键证据或台词释放 -> 压迫者表情崩塌 -> 被压迫者微表情反控`

## 竖屏构图

每条 `prompt` 必须写明 `9:16竖屏构图` 或 `超纵深垂直构图`。

- 人物脸部表演放在画面上中部或中轴偏上区域。
- 眼睛接近竖屏上三分线。
- 胸口以上特写保留头顶和下巴安全距离。
- 手部动作与关键道具保留在胸前或下中部安全区。
- 避免用横屏大片式远景铺陈替代人物冲突。
- 双人关系不得采用左右对称、平分画面的站位。竖屏短剧优先使用前后纵深、一坐一站、高低压迫、桌面/门框/窗框/道具阻隔、过肩遮挡、视线俯仰或关键道具错位来形成非对称构图。
- 人物近景、特写、大特写使用推进或变焦时，只需保证人物不是纯正脸看镜头。两人单独对话的面部镜头优先写轻微侧角或微侧脸，视线落在对手、文件、信封、货物、门口、地面或画外目标上，眼睛不看镜头。不要写正脸对镜头缓慢推进，除非剧本明确是自拍视频、直播、监控口供或主观镜头。

## 景别偏好

优先使用：

- 中近景。
- 近景。
- 特写。
- 大特写。
- 过肩镜头。
- 反打镜头。
- 插入镜头。
- 门框/窗框构图镜头。

竖屏短剧不得出现群像镜头，不得把多人同时作为镜头主体。涉及人群时，只能用前排单个路人近景/中近景、手部或肩背局部、虚化前景遮挡、现场议论声、主角反应反打来表现。

全景只用于交代站位、权力关系、人物进退和空间阻隔，且不得把人群作为主体。大远景只在剧本确实需要宏观地点或命运感时使用。

## 人物入场

- 任何角色首次出现在当前画面、从画外进入场景、或从上一组未入镜状态转为入镜状态时，必须先写清可见的入场过程，不能直接出现在最终剧本位置。
- 入场过程必须绑定场景来源、方向和遮挡关系，例如从门后、走廊、楼梯口、车门外、桌侧、屏风后、窗边、虚化肩背后、画面左/右边缘或画外脚步声方向进入；不要只写“@角色 出现”“@角色 站在门口”“@角色 已经站到桌前”。
- 竖屏真人短剧的人物入场优先使用 `遮挡后显露` 从门框、墙角、货架、肩背或窗框后带出人物；也可使用 `横向左移/横向右移` 揭示门口来人，`跟随式跟拍` 或 `引导式跟拍` 跟随角色进入空间，`第一人称视角行走` 表现靠近，`由虚到实` 从门口虚焦切实到人物轮廓。
- 入场镜头通常1到3秒，必须有明确起点和落点：脚步声/门缝/手推门/身体越过门框/停在某个场景锚点旁。入场完成后，后续镜头或下一组的 `【空间构图与人物位置】` 才能写角色的稳定站位、朝向和视线关系。
- 如果角色已经在上一镜或上一组清楚入镜且位置连续，后续不需要重复入场，但必须保持站位、朝向、持物和视线关系连续。

## 反应镜头

高能台词、身份揭露、背叛、亲密试探、羞辱、误会解除、复仇爽点、死亡威胁之后，不可只拍说话者。必须至少加入一个倾听者或被冲击者的独立反应镜头。

独立反应镜头必须紧凑。普通无对白反应、单一表情特写、倾听者特写优先1到2秒；只有一级爆点、身份揭露、关键证据落点或表情从起势到余韵有明确变化时，才可写3秒。

反应镜头必须包含可见表演：

- 眼睑微动。
- 瞳孔收缩。
- 唇部抿紧或微张。
- 下颌绷紧。
- 呼吸停顿。
- 喉结滚动。
- 指尖蜷缩或松开。
- 肩颈收缩。

重要反应镜头要有微表情弧线：

`起势：眼睑微动/呼吸一滞/唇部轻抿 -> 爆点：瞳孔收缩/泪光聚集/下颌绷紧 -> 余韵：视线偏移/气息缓出/表情凝固/手指松开`

## 台词与切镜

- 普通短句对白优先在同一镜头内完成，保持演员表演连续性。
- 关键台词前后的铺垫和反应，可以合并进同一镜头的起幅、停顿或收尾，不必默认拆成三镜。
- 台词前后的轻微心理/表情提示必须并入紧邻台词镜头，例如“环顾四周，面露嫌弃”后立刻说话时，应写在同一个说话者镜头的起幅中，不得单独拆成一镜。类似“似笑非笑、皱眉、冷笑、看了一眼、停顿半拍”等只在推动信息、道具落点或权力关系变化时才可独立成镜。
- 台词所在镜头聚焦口型、眼神和脸部肌肉；若同镜头包含对方反应，必须写清视线、呼吸或手部变化。
- 台词后一镜只在一级爆点、关键道具落点或关系反转时单独给对方反应或道具余韵。普通答话、普通停顿、普通看一眼不得在组尾额外补反应或空镜。
- 单个含台词镜头必须先按可朗读字数核算时长，再决定是否拆句。普通中文对白按每秒5到6个汉字估算；1到6个汉字通常1秒，7到12个汉字通常2秒，13到18个汉字通常3秒。
- 单个3秒对白镜头最多承载18个汉字；超过18个汉字必须按原文语义拆成多个连续台词镜头，或给足4秒以上的真实朗读时长，但仍受单组15秒上限约束。
- 不得把超过18个汉字的台词写成3秒镜头；不得因为想保持同一人物连续表演，就牺牲口播可读时长。
- 台词包含动作转折、情绪转折、信息反转、身份揭露或关键道具落点时，即使不超过18个汉字，也可以按表演节奏拆成多个镜头。
- 竖屏短剧中，长对话、超过3秒的台词或超过18个汉字的3秒台词不得用双人中近景或多人关系镜头硬撑到底，且任何位置都不得使用群像镜头。必须优先拆成说话者近景/特写、衣服/文件/手机等关键道具插入、有效环境承接、倾听者有效反应；双人关系镜头只用于短暂建立站位、关系压迫、动作调度或台词前后的空间交代。
- 长对话跨组时，下一组开头必须与上一组结尾画面错开并顺畅衔接。竖屏短剧每组第一个镜头不得使用反打镜头作为起手；组首优先拍当前说话者动作、手部动作、新落点、过肩关系、有效道具或入场动作。若上一组最后镜头是说话者特写，下一组开头可以用对方的有效动作、关键道具或手部动作承接画外音，但不得为了错开而添加无意义空镜。
- 独立反应镜头只用于一级爆点；普通倾听反应写入同框、过肩或说话者镜头的画面后半段。

## 用户分镜吸收规则

- 用户提供更细的分镜时，优先吸收“画面更明确”的内容：场景锚点、桌面/门口/照片/样衣/手机等道具、正品与仿品等对照关系、人物视线移动、手指先后指向、表情从紧到松或从迟疑到笃定的可见弧线。
- 用户提供的单镜时长只作为参考，不得照抄为硬约束。吸收分镜后必须重新按“台词字数、动作链复杂度、信息推进强度”核算时长：短句对白和轻微起幅/收尾动作优先合并，普通中文7到12个汉字的对白通常2秒，叠加视线移动、轻微表情或开口前半拍通常不超过3秒。
- 普通道具交接、接过物品、手指收紧、视线从道具回到人物等轻量动作链，通常2到3秒完成；只有多步骤取出、递出、落点压住证据、焦点回到人物并产生新剧情信息的完整道具动作链，才允许超过3秒，但仍必须写清每一秒内的信息变化。
- 不照抄会降低生成稳定性的内容：一个镜头内写多次“镜头切换”、重复“摄影机捕捉面部和手部动作”、台词过长不拆、只为“眼神锐利/眉头舒展”额外加镜头。把这类内容改写为景别、运镜、焦段、动作落点和台词拆分。
- 当对照道具承载反转，例如“有线头的正品”和“没有线头的假货”，镜头必须至少一次明确拍到两者的可见差异或指向顺序，并在爆点台词后给对方反应或道具余韵。

## 微表情转译

禁止最终写“悲伤、愤怒、心虚、深情、震惊、崩溃、释然”等抽象情绪词，禁止使用比喻、拟人、象征、夸张、成语化或诗化修辞，必须转译为可拍摄细节。不要写“心如刀割、怒火中烧、万箭穿心、杀气腾腾、柔情似水、绝望如深渊、城市沉睡、时间流淌”等表达；改写为身体姿态、呼吸、肌肉、眼神、手指、口型、环境物件运动等字面画面。

- 震惊：瞳孔骤然收缩、眼睑瞬间撑开、呼吸停住、唇部微张后僵住。
- 隐忍：下颌线绷紧、唇部抿紧后松开、喉结滚动一次、指尖缓慢蜷缩。
- 恐惧：瞳孔明显扩大、眼白暴露增多、胸口快速起伏、手指轻微颤抖、身体向后缩紧。
- 愤怒：下颌肌肉隆起、颈部青筋可见、鼻孔扩张且呼气粗重、拳头握紧至指节发白、肩膀向前顶出。
- 悲伤：眼睑缓慢下垂、下眼睑湿润反光、嘴角明显下拉、肩膀向内收紧、头部低垂至胸口。
- 喜悦：嘴角明显上扬、眼角出现皱纹、身体舒展打开、动作幅度增大。
- 心虚：视线短暂躲闪、眼睑快速眨动、嘴角抽动、手指无意识摩挲衣角。
- 怀疑：眼神从对方眼睛移到手部或道具、眉峰轻压、唇角轻微下沉。
- 决绝：下颌向前顶出、眼神直视固定一点、嘴唇抿成一条线、动作变慢但用力加重。
- 麻木：面部松弛无表情、眨眼间隔变长、身体长时间静止、胸口起伏很弱。
- 绝望：瞳孔涣散无焦点、头部无力下垂、双手自然垂落身侧、膝盖弯曲或身体下坠。
- 仇恨：眼神直盯目标、嘴角下拉、下颌咬紧、呼吸平稳控制、动作精准无多余。
- 爱恋：视线缓慢扫过对方面部、眼睑低垂、嘴角轻微上扬、身体不自觉倾向对方。
- 心动：眼神停留延长、下眼睑微微放松、唇角极轻上扬、呼吸变轻。
- 崩溃：泪光迅速聚集、下颌失控颤动、肩膀微塌、气息断续。
- 反控：眼神重新聚焦、嘴角微不可察上扬、下颌线放松后稳定、指尖停止颤动。

情绪曲线必须连续，不得突然跳变。根据剧情选择正向链或负向链的一部分：正向链可从平静、疑惑、震惊、狂喜、失控、崩溃到释然中截取；负向链可从压抑、愤怒、爆发、疯狂、毁灭到空洞中截取。每个阶段必须通过表情、动作、声音至少两类同步体现；相邻阶段之间必须有过渡动作，例如呼吸卡住、手指收紧、视线偏移、肩膀收缩、下颌加力或动作速度改变。

情绪峰值阶段必须伴随可见肢体失控或力度变化，例如手指颤抖、拳头砸下、身体后缩、膝盖弯曲、肩膀抽动、动作突然变快或变重。情绪回落阶段必须通过呼吸节奏放缓、肌肉松弛、动作减速、视线下移或手指松开体现。

普通镜头最多写1到2个微表情；爆点特写最多写3个。每个微表情必须与台词、动作或情节信息有因果关系，禁止堆叠无因果关系的生理反应。

## 运镜、速度与焦距

每个镜头必须同时指定景别、`camera-rules.md` 的 `AI运镜手法库` 原名、速度/幅度和焦距或镜头属性。常用组合：

- 揭示式推镜：使用 `慢速推进` 或 `平滑向内变焦`，广角或标准镜头，用于命运逼近或发现关键细节。
- 聚焦式推镜：使用 `快速推进` 或 `平滑向内变焦`，中长焦或长焦，用于重大决定、情绪临界点和关键台词。
- 疏离式拉镜：使用 `慢速后拉` 或 `平滑向外变焦`，广角或超广角，用于被抛弃、失去或结尾孤立。
- 紧迫式跟镜：使用 `引导式跟拍`、`跟随式跟拍` 或 `平行侧跟`，广角，用于追逐、逃离或情绪爆发后的疾走。
- 攻击式甩镜：使用 `快速甩镜`，标准或中长焦，用于突然揭示新信息。
- 垂直升降：使用 `垂直上移` 或 `垂直下移`，用于竖屏高度关系、门框、楼梯、台阶和人物压迫关系。

竖版短剧优先从运镜库中使用 `慢速推进`、`快速推进`、`快速甩镜`、`手持纪录片`、`垂直上移`、`垂直下移`、`引导式跟拍`、`跟随式跟拍`、`过肩镜头`、`焦点切换`。需要稳定读口型、反应或道具时，才可写 `固定镜头/固定特写（静态例外）`，同一组最多2次。

推进和变焦拍人物脸时，必须避开正面直视镜头：两人单独对话只需写轻微侧角或微侧脸，视线看向对手、文件、道具、门口、地面或画外目标；不要让人物正脸看向镜头。若使用 `慢速推进`、`快速推进`、`平滑向内变焦`、`急速变焦` 承载台词或反应，镜头描述必须包含“眼睛不看镜头”或等价的明确视线落点。

焦段使用必须克制：普通对白、普通反应和双人关系镜头优先使用35mm、40mm、50mm或“标准镜头/中焦”；70mm只用于情绪明显压近、权力关系变化或关键台词近景；85mm只用于爆点特写、眼神特写、道具插入、手部动作和关键表情，不得作为普通对白反打的默认焦段。同一组内如无明确爆点，85mm最多出现1次；连续普通对白不得连续使用70mm以上焦段。

## 光影与质感

光源必须来自场景内合理来源：

- 客厅/家宴：暖黄吊灯或落地灯，桌面反射光补脸。
- 办公室/董事会：冷白顶灯加窗侧光，玻璃反射形成线条。
- 医院/病房：冷白柔光、浅蓝阴影、床头灯弱暖光。
- 酒店/宴会厅：暖色吊灯、局部轮廓光、玻璃杯和金属道具高光。
- 地下车库/走廊：顶灯分段照明，背景保持纵深。
- 车内夜景：仪表盘弱光、窗外霓虹反射、侧脸轮廓光。

近景、特写、大特写中必须保留眼神光和口型可读性。真人质感不追求过度磨皮，面部特写可写“保留真实皮肤纹理”“浅景深柔化背景但面部焦点清晰”。

## 空镜与插入镜头

空镜只能用于承接重台词、强调关键物品、制造留白或完成景别过渡。连续空镜不得超过2个，不可替代人物反应镜头。无人物动作、无台词声音、无关键证据读数、无关系变化的状态空镜必须删除。每组结尾不得添加没有意义的人物反应空镜；没有新的剧情信息时，组尾直接停在最后一句台词或动作结果上。

插入镜头用于戒指、协议、手机、诊断单、门把手、杯沿、伤口、纽扣、照片、钥匙等关键道具。道具占据画面主体，手部动作只保留一个核心动作。只有道具正在被拿起、递出、按下、确认、暴露信息或改变人物关系时才单独成镜；普通设备状态和环境陈设不单独成镜。插入镜头必须写清道具名称、道具所在位置、人物手部动作、视线或声音如何指向该道具，以及它提供的剧情信息；写不清这些内容时，不要使用插入镜头。

竖屏短剧中的空镜和插入镜头不要默认静止。承接重台词、人物出场或长对话拆分时，优先使用合适的运镜库原名：`遮挡后显露` 从门框、货架、虚化肩背后带出人物；`由虚到实` 从布料、文件、货箱或门口建立注意力；`焦点切换` 从证据、信封、出货单切到人物侧脸或反应；`垂直下移` 从脸落到手部/道具，`垂直上移` 从道具升到侧脸；`慢速后拉` 或 `平滑向外变焦` 揭示空间压力；`横向左移/横向右移` 揭示门口来人、文件位置或样衣差异。空镜通常1到2秒，只有承接重台词、关键证据读数或完整道具动作链时才写3秒。
